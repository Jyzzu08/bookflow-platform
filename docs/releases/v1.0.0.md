# Release Notes v1.0.0

## Highlights

- Legacy `v1` compatibility endpoints preserved behind the gateway.
- Modern `v2` routes available for auth, catalog, comments, orders, payments, and search.
- Catalog migrated to Laravel and validated inside the active Docker Compose stack.
- Full local stack available through Docker Compose.
- Observability baseline included with Prometheus, Grafana, Loki, Tempo, and OpenTelemetry.
- Validation flow included with smoke, contract, and end-to-end scripts.
